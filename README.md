A Secure Folder Locker

A powerful, easy-to-use desktop application that completely hides and protects your sensitive folders and files with strong encryption.

When you lock a folder or file:
- It disappears from its original location
- Moves to a hidden encrypted vault
- Can only be accessed through this app with the correct password
- Auto-relocks when you close the window

Perfect for protecting private documents, photos, videos, or any sensitive data.

## Features

- Protect both **folders** and **individual files**
- Each item has its own password
- Master password protects access to the app
- Auto-relock when you close the Explorer/Finder window
- Progress bar for large operations
- Force Remove for stuck items
- Clean, modern dark interface
- No internet required · Fully offline
- Windows, macOS, Linux compatible (core features)

## Security

- AES-256-GCM encryption (military-grade)
- PBKDF2 with 120,000 iterations
- Passwords never stored in plain text
- Encrypted data is unrecoverable without password

## How to Use

1. Run `SecureFolderLocker.exe`
2. Set a master password on first launch
3. Click **Add Folder** or **Add File** to protect items
4. Enter a strong password for each item
5. The item disappears and is now hidden
6. Click **Unlock** → enter password → item appears and opens
7. Close the window → it hides and locks again automatically

## Settings

- Change master password
- Change password for any individual item
